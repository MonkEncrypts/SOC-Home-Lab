
# Event Field Analysis

This document explains the key Sysmon Event ID 1 fields observed during the execution of the PowerShell Encoded Command detection.

| Field | Observed Value | Investigation Value |
|---------|----------------|---------------------|
| Event ID | 1 | Indicates a Process Creation event generated by Sysmon. |
| Image | powershell.exe | Identifies the executable that was launched. |
| CommandLine | powershell.exe -EncodedCommand ... | Shows exactly how PowerShell was executed and reveals suspicious command-line parameters. |
| User | WIN10-SOC-ENDPOINT\SOCUser | Identifies the user account responsible for the execution. |
| ProcessGuid | Present | Provides a unique identifier that can be correlated with other Sysmon events such as network connections or process termination. |
| ParentImage | cmd.exe | Identifies the parent process that launched PowerShell, allowing analysts to reconstruct the process tree. |
| ParentProcessGuid | Present | Enables correlation between parent and child processes during incident investigation. |
| IntegrityLevel | Medium | Indicates the privilege level of the executing process. |
| Hashes | SHA256, SHA1, MD5 | Verifies the identity and integrity of the executable used during execution. |
| CurrentDirectory | C:\Users\SOCUser | Indicates the working directory from which the process was launched. |
